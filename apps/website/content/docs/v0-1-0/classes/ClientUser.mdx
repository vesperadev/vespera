### setUsername

▸ **setUsername**(`username`): [`Promise`]( https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise )\<[`ClientUser`](ClientUser)\>

Sets the username of the logged in client.
Changing usernames in Discord is heavily rate limited, with only 2 requests
every hour. Use this sparingly!

#### Parameters

| Name | Type | Description |
| :------ | :------ | :------ |
| `username` | `string` | The new username |

#### Returns

[`Promise`]( https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Promise )\<[`ClientUser`](ClientUser)\>

**`Example`**

```ts
// Set username
client.user.setUsername('discordjs')
  .then(user => console.log(`My new username is ${user.username}`))
  .catch(console.error);
```
